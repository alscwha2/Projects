1. Tooltips (20 pts)

Using the provided visualization, add tooltips to the choropleth map. The tooltip should show the county name and the population of that county when a user moves the mouse over that county. For example, "Milwaukee: 956406". You may either use the title attribute of an SVG element or create a separate div element that is positioned based on the mouse cursor and can be styled (or take multiple lines). For this part, you may also use an external library like d3-tip. The population for each county is stored in a d3.map named popLookup which is passed to the addTooltip function.

Hints:

  • The county name is stored in d.properties.county_nam
  • The population can be accessed via the popLookup.get function and passing the county name since popLookup is a d3.map.

2. Focus+Context (30 pts)

In the provided visualization, modify the bar chart to provide a cartesian distortion for only the x-axis (not the y-axis). As the mouse moves from left to right, the bars nearest the cursor should widen while those farther away should be compressed. Use Mike Bostock's fisheye.js library for this. See his page for details about the various distortion methods. Note that fisheye.js is already loaded by a4.html for your convenience.

Hints:

  • You only need to distort the x-axis!
  • Note that the first example on Bostock's page shows radial distortion. The scatterplot further down shows cartesian distortion.
  • The width of a rectangle can be computed by determining where the end points would be under distortion and then subtracting.
  • To move both the bar and text, consider changing the translation of the group (g element) that contains the rect and text.

3. Linked Highlighting (30 pts)

Link the highlighted bar to its corresponding location in the county map. Similarly, link the selected county to its corresponding location in the bar chart.

Hints:

  • In a bar, the name of the county can be accessed via the County attribute of the data.
  • D3 provides the .classed method on a selection to add or remove a class from the class attribute.
  • Use mouseenter and mouseout events to determine when the mouse enters a particular region.
  • filter will be very useful here.
  • Use d3.select(this) to obtain the visual element wrapped for use with D3.
  • To access the data corresponding to a visual element, .datum() will be useful.

Extra Credit: Histogram (20 pts)

Create a histogram binning the counties by population. You should use a logarithmic scale (log base 2).
